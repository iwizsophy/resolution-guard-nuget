# Security Policy

## Scope

This document is for security vulnerabilities only.

For general usage questions and non-security support, see `.github/SUPPORT.md`.

## Reporting a Vulnerability

Please do not report security issues in public issues or discussions.

Use GitHub private vulnerability reporting:

- https://github.com/iwizsophy/resolution-guard-nuget/security/advisories/new

General bugs and feature requests should be reported through normal issues (see `.github/SUPPORT.md`).

If private reporting is unavailable, open a temporary issue with only:

- "`Security report requested`"

and no technical details. Maintainers will move communication to a private channel.

## Disclosure Policy

- Coordinated disclosure after a fix is available
- Security advisories and release notes will include mitigation and affected versions
