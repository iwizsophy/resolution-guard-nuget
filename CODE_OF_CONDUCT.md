# Code of Conduct

## Purpose

This project aims to be a respectful and practical place for collaboration.
Everyone participating in this repository is expected to follow this Code of Conduct.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code and technical outcomes.
- Assume good intent, and ask clarifying questions before escalating.
- Accept that maintainers and contributors are volunteers.

## Unacceptable Behavior

- Harassment, hate speech, discrimination, threats, or intimidation.
- Personal attacks, insults, trolling, or repeated hostile comments.
- Doxxing or sharing private information without consent.
- Repeated excessive demands, entitlement behavior, or pressure for unpaid labor.
- Spamming issues/PRs/discussions after requests to stop.

## Scope

This policy applies to repository spaces, including issues, pull requests, code reviews, and discussions related to this project.

## Enforcement

Maintainers may take actions considered appropriate and proportionate to the behavior, including:

- Requesting changes in behavior
- Removing comments or content
- Closing conversations
- Temporarily or permanently restricting participation

## Reporting

If you experience or witness unacceptable behavior, report it privately when possible.

If private reporting is not available, create a temporary issue with only:

- "`Code of Conduct report requested`"

and no details. Maintainers will move follow-up to a private channel when possible.
